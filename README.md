# Excel-Finance-Tracker-
An advanced Excel-based Finance Tracker dashboard designed for data analysis. It automates expense tracking, income summaries, KPI calculations, and category-wise insights using formulas, Pivot Tables, and charts to support budgeting, trend analysis, and data-driven financial decisions.

🚀 Key Features

Automated income & expense tracking
Category-wise financial analysis
Monthly and yearly summaries
KPI metrics (Savings Rate, Total Spend, Net Balance)
Interactive dashboard with charts & slicers
Data validation for clean data entry

🛠️ Tools & Skills Used

Microsoft Excel
Pivot Tables & Pivot Charts
Advanced Formulas (SUMIFS, COUNTIFS, IF, VLOOKUP/XLOOKUP)
Conditional Formatting
KPI Design & Dashboarding
Data Cleaning & Structuring

📂 Project Structure

Raw Data Sheet – Transaction-level financial data
Cleaned Data Sheet – Structured and validated dataset
Pivot Analysis Sheet – Summary tables and KPIs
Dashboard Sheet – Visual insights and interactive reports

📈 KPIs Tracked

Total Income
Total Expenses
Net Savings
